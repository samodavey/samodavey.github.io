Coding Test
This is a coding test in which a user is able to register and log into a system in which they can access a library of books which they can use CRUD services alongside being able to download the tables in an Excel or CSV format.
Getting Started
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.
Prerequisites
What things you need to install the software and how to install them
* Laravel
* Laravel Homestead (Covers a large variety of prerequisites)
* MYSQL
* MYSQL Workbench
Installing
A step by step series of examples that tell you how to get a development env running
Say what the step will be
1. Clone the repository to your local machine
2. Setup your local host
3. Make sure the .env file matches up with your local host (All other database parameters should be correct).
4. Begin using the service from localhost (127.0.0.1:8000)
